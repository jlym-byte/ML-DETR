# Model Configs

This directory is intentionally split by model family.

```text
ml-detr/
  ml-detr.yaml
  ml-detr-ecie.yaml
  ml-detr-ecie-toglsr.yaml

rt-detr/
  rtdetr-r18.yaml
  rtdetr-r34.yaml
  rtdetr-r50.yaml
```

Use `ml-detr/` for the proposed ML-DETR models and `rt-detr/` for RT-DETR baselines.
